# Contributing to Data Resource Bank

First off, thank you for considering contributing to the Data Resource Bank! It's people like you that make this repository a great knowledge base.

## How to Contribute
1. **Fork** the repository on GitHub.
2. **Clone** your fork locally.
3. **Draft** your new resource using the `RESOURCE_TEMPLATE.md` to ensure consistency.
4. **Place** the new `.md` file in the appropriate category folder.
5. **Commit** your changes with clear messages (e.g., `Add Python beginner tutorial`).
6. **Push** your branch to your fork.
7. Open a **Pull Request** to the main repository.

## Formatting & Naming Conventions
- Keep folder and file names clean (use hyphens or underscores instead of spaces).
- Use `.md` extensions for documents.
- Keep titles descriptive but concise.

## Resource Quality Guidelines
- Ensure all links are active and functional.
- Please verify if a similar resource already exists before adding a new one.
- Make sure content remains beginner-friendly where possible, or clearly label its difficulty.
